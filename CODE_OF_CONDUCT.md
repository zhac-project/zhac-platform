# Code of Conduct

The ZHAC project has adopted the [Contributor Covenant][cc] (version 2.1)
as its Code of Conduct. Everyone who participates in the project — through
issues, pull requests, discussions, or any other project space — is
expected to uphold it, and to do the same in public spaces when
representing the project.

The full text is published here:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Report any concern to the project maintainer at
**zhac.project@gmail.com**. Reports are reviewed and handled
confidentially, and the maintainer will respond as promptly as is
practical. Maintainers are obligated to respect the privacy and security
of anyone who reports an incident.

[cc]: https://www.contributor-covenant.org/
